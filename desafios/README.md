Adicione os desafios em `desafios.js`.

Cada desafio deve ter este formato:

```js
{ name: 'Nome do desafio', url: 'https://link-direto-da-imagem.png' }
```

Na tela inicial, clique em "Sortear" para entregar um desafio para cada jogador.
O jogo mostra um QR code para o link da imagem, sem interferir na partida.
